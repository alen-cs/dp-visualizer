二进制拆分原理

任何一个数字都可以用 $1, 2, 4, 8...$ 等 2 的幂次方加上一个余数表示。从而将 $O(N)$ 复杂度降至 $O(\log N)$。

// 多重背包的二进制拆分核心 C++ / JS 代码示例 for(let i = 1; i <= n; i++) { let weight = items[i].w, value = items[i].v, amount = items[i].s; let k = 1; // 箱子容量 while(amount >= k) { newItems.push({ w: weight * k, v: value * k }); amount -= k; k *= 2; } if(amount > 0) newItems.push({ w: weight * amount, v: value * amount }); }
